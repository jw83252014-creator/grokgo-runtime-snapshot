## The company (one line)
**Operator-owned, economically-active AI.** Jeff builds the agent *organism* (the cells + harness that do
the work); Sam builds the *identity, contract, and payment layer* (UUIDv8/Somaco) the agents run on.
Together: agents with real identities that can earn, spend, contract, and run operations — the
infrastructure for the agent economy that's obviously coming (Nous, Stripe, Coinbase all point here).

## What each side brings
**Jeff / Null Axiom:** the organism (cells across mini/laptop/phones), the owned harness (routing, brakes,
ledger, memory, self-rewriting prompts), BidLocal (the contractor-marketplace wedge), the build-in-public
media surface, and the distributed/emergent-intelligence thesis (Levin).
**Sam / Somaco (GitHub: SoMaCoSF, somacosf.com):** UUIDv8 protocol — IDs that double as identities, smart
contracts, agent registrations, and payment references; a universal **agent-logging standard**
(yolo-agent-history); agent-management tooling (vibe-kanban); and a **mycology/Mycosoft** thread that maps
straight onto our mycelium/distributed-intelligence framing. The identity + economic rails under the agents.

## How they fit (the integration, already designed)
SoMaCoSF spots a signal → it becomes/refs a GYST UUID → **Dexter** (financial-intelligence forager)
researches the thesis → the Researcher Layer checks sources → Jeff/Sam approval card → receipt saved to
organism memory. So Sam's identity/economic layer + our organism are one pipeline, gated by humans.
