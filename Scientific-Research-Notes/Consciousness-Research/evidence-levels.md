# Evidence Levels For Consciousness / Psi Notes

Use one or more labels per claim.

| Label | Meaning | Public Copy Rule |
| --- | --- | --- |
| `mainstream_supported` | Supported by well-established science | Can state plainly with source |
| `limited_early` | Some studies or partial evidence, not settled | Use cautious wording |
| `anecdotal` | Personal experience or testimony | Attribute as personal report |
| `speculative` | Hypothesis or synthesis beyond evidence | Mark as speculation |
| `symbolic_metaphor` | Mythological/archetypal framing | Present as metaphor, not fact |
| `unverified` | Not yet checked against source | Do not present publicly |
| `unsupported_or_conflicting` | Conflicts with mainstream evidence or lacks support | Avoid factual claim; can discuss as belief/framework |

## Default Public Safety Line

When drafting content from this section:

> I would frame this as a symbolic/protocol idea people are exploring, not as settled science.
